# Process (GNU `coreutils`)

| Command | Comment |
| --- | --- |
| `kill` | Send a signal to processes |
| `sleep` | Delay for a specified time |
| `timeout` | Run a command with a time limit |
| `nohup` | Run a command immune to hangups |
